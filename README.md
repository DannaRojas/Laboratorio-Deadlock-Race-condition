#            Laboratorio Deadlock, Race-Condition, Hilos POSIX y Mutex 

## Descripción 
Este repositorio contiene una serie de programas desarrollados en C para el Laboratorio de eadlock, Race-Condition, Hilos POSIX y Mutex 

El objetivo principal es aplicar los conceptos de paralelismo y sincronización utilizando hilos POSIX (pthread) y mecanismos como mutex y variables de condición.

## Compilación
Para compilar los programas, utiliza el comando:

make nombre_programa

Por ejemplo:

make hilos_pthreads03Rojas

Esto generará un ejecutable con el mismo nombre. Y para limpiar los ejecutables generados:

make clean
